# Japanese Craft Marketplace

A Japanese craft marketplace built with TanStack Start, React, Vite, Tailwind CSS, and TypeScript.

## Requirements

- Node.js 18+ or Bun

## Install

Using Bun:

```bash
bun install
```

Using npm:

```bash
npm install
```

## Run locally

Start the development server:

```bash
bun run dev
```

If you prefer npm:

```bash
npm run dev
```

## Build

Create a production build:

```bash
bun run build
```

Or with npm:

```bash
npm run build
```

## Preview production build

```bash
bun run preview
```

Or with npm:

```bash
npm run preview
```

## Lint

```bash
bun run lint
```

Or with npm:

```bash
npm run lint
```

## Scripts

- `dev` - Start the Vite dev server
- `build` - Build the app for production
- `build:dev` - Build using development mode
- `preview` - Preview the production build locally
- `lint` - Run ESLint
- `format` - Format files with Prettier
